About Me

I am a passionate researcher, proficient in developing end-to-end Machine Learning/Deep Learning applications with focus towards designing hardware-efficient algorithms, optimizing latency and power-consumption as much as possible. I am particularly interested in exploring brain-inspired Spiking Neural Network models and algorithms for sensor-based applications. Besides, I have strong research experience in Computer Vision (Image Classification/ Segmentation, Object Detection), NLP (ASR, Text Processing, Speech-to-Text Conversion) and Computer Systems (Computer Architecture, Embedded/Wireless Systems).

Projects

Event-based TNNs for gesture recognition, Neuromorphic Computer Architecture

Devised multiple encoding schemes to feed the event-based DVS data into the custom Temporal Neural Network (TNN) architecture. Implemented spike-time encoding algorithm using multiple bit levels of quantization on NMNIST dataset. Built an adapted 2D point cloud encoding of events to create patterns on the spatial and temporal waveform of events.

Delta RNN – GRU on a Silicon Cochlea – Spoken Digit Recognition

Built a custom Delta GRU from scratch to increase sparsity in matrix vector multiplication components of the weight matrices. Implemented Delta GRU on the silicon cochlea – an eight-channel filtered output of a spoken digit audio wave. Implemented 2 x 2 Spiking Neural Network on Cadence using Muller-C element and 2-input arbiter.

Real-time Lane Identification System with GANs

Built a 2-layer unidirectional CNN-LSTM architecture in PyTorch to localize a car to its lane, on time-series accelerometer data. Employed data augmentation, GAN on 1D sequence data to generate acc(Z) readings for different road conditions/ car types.

Drive Smart - Low Power Driver Distraction Identification and Alert System

Built a low power system using Raspberry Pi to monitor driver’s attention in a car and send inattention alerts via piezo buzzer. Selectively enabled the system via interrupts & polling to minimize power, using proximity sensor to detect driver’s presence. Implemented face detection, feature extraction (eyelids, nose length, jaw distance) w/ OpenCV Haar Cascade Classifier & dlib.

Neural Predictions on Human Activity Recognition

Implemented sequence classification by segmenting sensor signals, applying multi-head & self-attention - tensor2tensor. Performed activity recognition - employing backpropagation on LSTM w/ encoder of transformer model, calculated F1 score.

Optimal Level-Crossing Prediction for Electronic Prognosis

Evaluated optimality of failure prediction methods on battery data - RMSE on Gaussian Process Regression & Random Forest. Applied state space approach for feature design and Kalman filtering via MATLAB to enhance linearity in the capacitance data.
